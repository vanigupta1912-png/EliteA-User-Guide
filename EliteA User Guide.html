Introduction to Elitea Nexus

Welcome to the Elitea Nexus User Guide! This guide will help you sign in, explore the platform, and get the most out of EliteA — EPAM's AI-powered assistant.

What is Elitea Nexus?

Elitea Nexus is the secure platform that hosts the EliteA AI assistant and a full suite of workspace tools. It is designed for enterprise teams who need intelligent automation, structured collaboration, and fine-grained access control — all in one place.

What is EliteA?

EliteA is the AI assistant embedded within Elitea Nexus. It can help you answer questions, analyse documents, draft content, troubleshoot technical issues, and automate multi-step tasks through agents and pipelines.

Who is it for?

  • Software engineers and developers
  • Business analysts and project managers
  • Technical writers and documentation teams
  • DevOps and platform engineers
  • Any enterprise user needing AI-assisted productivity

Key Benefits

🤖 AI-Powered Chat

Ask EliteA anything — from code help to document summaries — in a natural conversation.

🔒 Enterprise Security

OIDC authentication, role-based access, and encrypted conversations keep your data safe.

Build and run intelligent agents and multi-step automated pipelines for complex workflows.

Monitor conversation context budgets so you always know how much space remains in a session.

Core Features

Discover the essential features that power your day-to-day work in Elitea Nexus.

💬

Chat with EliteA

The Chat section is your primary space for AI-assisted conversations.

How to use:

  1. 1Select Chat from the left sidebar
  2. 2Type your question or request in the message composer at the bottom
  3. 3Optionally type @ to mention an assistant or # to search assistants
  4. 4Press Enter or click the send arrow to submit
  5. 5EliteA responds in the main chat area; continue the conversation naturally
💡 Tip: Be specific and include context. For example: “Summarise this error log and suggest the top 3 fixes” yields much better results than “fix error”.
📁

Conversations & Folders

How to use:

  1. 1Click + Create to start a new conversation or folder
  2. 2Name your folder to group related conversations together
  3. 3Use Search in Chats to quickly find past conversations by keyword
  4. 4Click any conversation in the list to continue it
💡 Tip: Organise conversations into folders by project or topic for faster access.
🧹

Clear Chat History

How to use:

  1. 1Click the Clear chat history button in the bottom-right panel
  2. 2Confirm the action when prompted
  3. 3The conversation context is cleared, giving EliteA a fresh start
⚠️ Note: Clearing chat history is permanent. Ensure you have saved any important outputs before proceeding.

Best Prompt Style

A good prompt includes four elements:

  1. 1Your goal — what do you want EliteA to do?
  2. 2Relevant context — background information EliteA needs
  3. 3File or text input — paste code, logs, or attach a document
  4. 4Preferred output format — bullet list, table, code block, paragraph, etc.

Example prompts:

"Summarise this report in 5 bullet points."
"Analyse this login error and suggest fixes."
"Draft a short onboarding guide for new users."
"Compare REST vs GraphQL and recommend the best option for a mobile app."
💡 Pro tip: Use @assistant-name in the chat to route your message to a specific AI assistant configured for that task.

Troubleshooting

Encountering an issue? Find the most likely cause and solution below.

Login Errors

HIGH

Solutions:

  1. 1Double-check your username/email and password for typos
  2. 2Use the password reset option on the login page if needed
  3. 3If using OIDC, confirm your identity provider is correctly configured
  4. 4Contact your IT administrator for OIDC-related access issues
  5. 5Try a different browser (Chrome or Firefox recommended)

OIDC / SSO Authentication Issues

HIGH

Solutions:

  1. 1Confirm your enterprise identity provider (EPAM or Forefront) is set up correctly
  2. 2Ensure your account has been provisioned in the identity provider
  3. 3Clear browser cookies and cache, then try again
  4. 4Contact your IT administrator to verify OIDC configuration

Dashboard or Chat Not Loading

MEDIUM

Solutions:

  1. 1Check your internet connection
  2. 2Refresh the page (F5 or Ctrl+R)
  3. 3Disable browser extensions that may interfere
  4. 4Try an incognito/private window
  5. 5Contact EliteA support if the issue continues

Slow or No AI Response

MEDIUM

Solutions:

  1. 1Check the Context Budget — if near the limit, clear chat history and start fresh
  2. 2Break very long prompts into shorter, focused requests
  3. 3Check your network connection speed
  4. 4Refresh the page and try again

Context Budget Depleted

LOW

Solutions:

  1. 1Click Clear chat history in the right panel to reset the context
  2. 2Start a new conversation using + Create
  3. 3Avoid pasting extremely large documents in a single message; split them up

Inaccurate or Unhelpful Responses

LOW

Solutions:

  1. 1Add more context and be specific about your goal
  2. 2Rephrase your question using the four-element prompt structure (see How to Use)
  3. 3Specify the format you want: bullet list, table, code, etc.
  4. 4Try a different AI assistant using @ mention in the chat
